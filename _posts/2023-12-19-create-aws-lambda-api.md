# 개요

보통 우리가 개발한 서버를 띄우기 위해 Azure나 AWS 같은 곳에서 서버를 대여한다. 그런데 대부분의 서버 가격 책정은 시간단위로 측정한다. 즉, 아무도 사용 하지 않음에도 불구하고 비용이 계속 든다는 점이다.
하지만 AWS Lambda는 사용한 횟수만큼 비용을 책정하기 때문에, 아무도 사용하지 않는 기간 동안은 가격이 책정되지 않는다는 장점이 있다.
뿐만 아니라 서버 환경 세팅은 AWS Lambda 쪽에서 알아서 해주기 때문에 API 구현 말고는 배포를 위한 다른 작업을 할 필요가 없다.
